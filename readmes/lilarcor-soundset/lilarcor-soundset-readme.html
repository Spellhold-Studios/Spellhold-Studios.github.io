Introduction

    This is Lilarcor, the homicidal talking sword with a one-track mind. You know him, you love him! Now available as a soundset for an equally homicidal player character with a one-track mind, with full support for thieving skills and everything.

    Thoroughly tested by yours truly. I did my best to avoid annoying repetitions and I think it turned out rather well, though Lilarcor obviously isn't for everyone. I tested with a Thief, which is important because an annoying Hid In Shadows sound gets old really fast if you use a script that keeps your character hidden.

Installation

  1. Put the files in the sounds folder in your Baldurs Gate 2\sounds folder.
  2. Go to Character Record -> Customize -> Sounds, and select LILARCO.
  3. Enjoy!

    Uninstallation

      Just delete the files.
