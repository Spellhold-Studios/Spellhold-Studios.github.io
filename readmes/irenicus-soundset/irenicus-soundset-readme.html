Introduction

    This is Jon Irenicus, the megalomanic, arrogant bad guy we all know and love, voiced by none other than David Warner. Now available as a soundset for an equally megalomanic, arrogant player character, with full support for thieving skills and everything.

    Thoroughly tested by yours truly. I did my best to avoid annoying repetitions and at the same time pay proper homage to the classics. I think it turned out rather well, though Jonny didn't offer a whole lot of options for some of the sounds. I went through each and every Irenicus sound file in the game, rather meticulously. I tested with a Monk, which is important because an annoying Hid In Shadows sound gets old really fast if you use a script that keeps your character hidden.

Installation

  1. Put the files in the sounds folder in your Baldurs Gate 2\sounds folder.
  2. Go to Character Record -> Customize -> Sounds, and select IRENICU.
  3. Enjoy!

    Uninstallation

      Just delete the files.
