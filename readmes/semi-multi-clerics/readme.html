Introduction

This mod will add various clerical spells (including one that can be used to turn undead) to wizard magic users via a new item, effectively turning them into previously unavailable X / Cleric kits / multiclass combinations.
The original readme is available for reference as well and contains information on the exact effects of the mod. In case of conflict, trust this document over the old readme.

Compatibility notes

For BGT / BG2 users, when Tobex is installed the item price is lowered to 1Po due to option Item Recharge Mod.

Can be disabled in an ongoing game will setting :

Item Recharge Mod=1 to Item Recharge Mod=0 in folder Baldurs Gate 2\TobEx_ini\TobExCore.ini.


Generally speaking, this mod should be compatible with virtually any other decently made WeiDU mod in existance. In over twenty years, no incompatibility has been reported.


  • For BG2, the mod can be used only if you have access to either Waukeen Promenade and /or the Pocket Plane; if you cannot, you can use the following CLUA console command to acces the store:
CLUAConsole:CreateCreature("tb#prsp")

  • If you are playing on BGT or EET, the seller can be found at Firebeard house, at the Sorcerous Sundries, in Waukeen Promenade and the Pocket Plane.

  • If you are playing on IWDEE or BGEE, the seller can be found near Orrik's tower or in Firebeard house or at the Sorcerous Sundries. (But you'll be allowed to buy the spell book only once you've reached 5000 XP.).

  • Merchant will also appear in hub areas from Black Pits I and II.

Installation

Windows

To install the mod, extracted the archive you downloaded to your BG2 installation directory using WinZIP, WinRAR, or the `Extract To...' wizard from Windows XP. If properly extracted, you should have a semi_multi_clerics folder, setup-semi_multi_clerics.tp2 and setup-semi_multi_clerics.exe in your BG2 folder. To install, simply double-click Setup-semi_multi_clerics.exe and follow the instructions on screen.

Please re-run Setup-semi_multi_clerics.exe in your BG2 folder to reinstall, uninstall or otherwise change components.

Macintosh

If the mod was previously installed, uninstall it before extracting the new version. Semi Multi Clerics is packaged and installed with WeiDU. To install, simply extract the contents of the mod into your BG2 folder. If properly extracted, you should have a semi_multi_clerics folder, setup-semi_multi_clerics.tp2, Setup-semi_multi_clerics, and Setup-semi_multi_clerics.command in your BG2 folder. To install, simply double-click Setup-semi_multi_clerics.command and follow the instructions on screen.

Linux

Download the latest version of WeiDU for Linux from weidu.org and extract (at least) the WeiDU, weinstall and tolower executables in a $PATH directory (E.G. /usr/bin).
Following that, open a terminal, cd to your BG2 installation directory, run `tolower' and answer Y to both queries.
You can avoid running the second option (linux.ini) if you've already ran it once in the same directory. To save time, the archive is already tolowered, so there's no need to run the first option (lowercasing file names) either if you've extracted only this mod since the last time you lower cased file name.
If you're unsure, running tolower and choosing both options is the safe bet.
After that, run 'weinstall semi_multi_clerics' from your BG2 installation directory to install or uninstall the mod.

Components

A new merchant named Mystra's Witness in the following positions:
  • ar0700, x1715 y1020 (Waukeen's Promenade, north of the Adventurer's Mart)
  • ar4500, x2250 y1350 (the Pocket Plane)
  • ar2100, x423 y901 (Kuldahar, Near Horrick Tower for IWDEE)
  • ar3333 / bg3333, x420 y189 (Beregost, Firebead house)
  • ar0703 / bg0703, x686 y201 (Baldur's Gate Sorcerous Sundries)
After talking to him, you can acquire a spell book that will teach you some divine spells (including one that can be used to turn undead), at the cost of some skill points and experience. (47000 xp, 2 Constitution and 2 Dexterite)

More details on the exact penalties and bonuses, along with thoughts on balance, are available in the original readme.

Version [ORIGINAL]

Accessible at the beginning / 100% chance of losing 2 points of Dexterite and Constitution.

Version [SOFT]

Accessible at the beginning / 25% chance of losing 2 points of Dexterite and Constitution

Version [ORIGINAL DELAYED]

Accessible after Underdark (bg2) or at Baldur's Gate (bg1) or after Dragon's Eye (Iwd) / 100% chance of losing 2 points of Dexterite and Constitution.

Version [SOFT DELAYED]

Accessible after Underdark (bg2) or at Baldur's Gate (bg1) or after Dragon's Eye (Iwd) / 25% chance of losing 2 points of Dexterite and Constitution.

Add a new reply when the store is empty

The merchant will simply tell you that it's done and leave.
