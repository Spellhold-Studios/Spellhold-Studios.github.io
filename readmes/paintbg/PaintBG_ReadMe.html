Introduction

PaintBG is a mod that replaces each of the NPC portraits of Baldur's Gate with original artwork by artastrophe. This portrait pack also overwrites the PC portraits included in the original game with new images to choose from! As always, each of these components is available to install separately.

Installation

PC

The PaintBG portrait pack is a WeiDU mod, packaged in an extractable .RAR archive. Simply unzip "PaintBG v.XY.rar" (WinRAR) into your Baldur's Gate II directory (usually "C:\Program Files\Black Isle\BGII - SoA" or "C:\Program Files (x86)\Black Isle\BGII - SoA")and run "setup-PaintBG.exe".

To uninstall, run "setup-PaintBG.exe" again and select the uninstall option.

Compatibility and Installation Order

We have done our best to test on every platform, and this mod should be compatible with all other mods (excluding those which change portraits, obviously: for example, wonnimchunha's Portrait Mod). Please report any suspected conflicts and we will do our best to make everything work together smoothly. If you do encounter an error, please post on the forums.

Although it is not required for the PaintBG Portrait Pack to function properly, it is always a good idea to install the latest version of the BG2 Fixpack.

Baldur's Gate: Enhanced Edition is unfortunately not compatible with WeiDU, and also uses different portrait dimensions. While you can't install the portraits using the .exe file, the resized components are still available to extract into your Override folder (usually "C:\Users\Documents\Baldur's Gate - Enhanced Edition")! Check the .RAR directory for the BG:EE-compatible sizes.
